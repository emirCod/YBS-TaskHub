# YBS TaskHub

A modern project and task management system developed with Flask, SQLite and Bootstrap.

## Features

- User authentication
- Dashboard with project statistics
- Create, edit and delete projects
- Create, edit and delete tasks
- Kanban board
- Project detail page
- Progress tracking
- Risky task monitoring
- Deadline management

## Technologies

- Python
- Flask
- SQLite
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

## Installation

Clone the repository

```bash
git clone https://github.com/KULLANICI_ADIN/YBS-TaskHub.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

## Database

The database schema is provided in **models.sql**.

Example data can be generated using

```bash
python seed_demo.py
```
